     ENTREGA FINAL - ENLACES DE REPOSITORIOS GITHUB
======================================================

Este proyecto está construido como un **Monorepo**. Todos los componentes solicitados se encuentran estructurados en directorios dentro del mismo repositorio.

1. **Repositorio Principal (Monorepo)**
   URL: https://github.com/Isaac2832821/proyecto-full-stack-3
   
   Estructura interna:
   - `/frontend` : Aplicación web cliente (Componente NPM React + Vite)
   - `/api-gateway` : Backend For Frontend (BFF) y API Gateway
   - `/ms-autenticacion` : Microservicio de seguridad y usuarios
   - `/ms-asistencia` : Microservicio de asistencia
   - `/ms-calificaciones` : Microservicio de calificaciones
   - `/arquetipos-maven` : Plantilla base para generación de microservicios
   - `/.github/workflows` : Pipelines de CI/CD para AWS EC2
