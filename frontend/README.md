# Frontend - Sistema de Gestión Escolar

Este es el cliente web desarrollado en React y Vite para la plataforma escolar.

## Requisitos
- Node.js (v16+)
- NPM

## Instalación y Ejecución Local
1. Instalar dependencias:
   ```bash
   npm install
   ```
2. Iniciar servidor de desarrollo:
   ```bash
   npm run dev
   ```

## Empaquetado para Producción
```bash
npm run build
```
Esto generará la carpeta `dist/` lista para ser servida (ej. con Nginx).
